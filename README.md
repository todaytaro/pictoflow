# PictoFlow

> AIで商品画像を、秒で量産。中小EC事業者向けAI画像生成SaaS。

## Overview

プロンプト不要。ブランドキットを登録するだけで、SNS・LP・バナー用の商品画像を10秒で量産できるSaaSです。

## Stack

- Frontend: HTML / CSS / Vanilla JS（MVP）→ Next.js（Phase2）
- Hosting: Vercel
- Image Generation: Flux.1 via Replicate API（予定）
- Auth & DB: Supabase（予定）
- Payment: Stripe（予定）
